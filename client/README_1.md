
# DatingApp - Kết nối trái tim.

Tech Stack: .NET 8/9, Entity Framework Core, SQL Server/PostgreSQL, Angular/React, AutoMapper, Cloudinary.

Features: Đăng ký/Đăng nhập (JWT), Upload ảnh, Nhắn tin thời gian thực, Phân trang & Lọc.

Local Setup: git clone, dotnet restore, dotnet ef database update, dotnet run.
